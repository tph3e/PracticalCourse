The simulator runs on Python with pm4py for log handling and the BPMN-to-Petri-net
conversion, pandas and NumPy for data handling, scikit-learn and SciPy for the
processing-time, branching and role-discovery models, and Matplotlib for figures. The
reinforcement-learning policy additionally uses gymnasium and sb3-contrib (MaskablePPO),
kept in a separate environment. OrdinoR is run offline to derive the role model. Random
seeds are pinned so the reported numbers reproduce. 

\textbf{Code, models and figures} are available in
the public repository \url{https://github.com/tph3e/PracticalCourse}.

\textbf{Generative-AI tools} were used for code review, debugging support, language
editing, and the preparation of test and evaluation scripts. All suggestions
were manually reviewed, adapted, and validated by the responsible group
members. Reported empirical results were produced by executable repository
artifacts and checked against the corresponding result files.
